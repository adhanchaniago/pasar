Pasar

Online marketplace skeleton written in ruby on Rails

This is an Online marketplace skeleton written in Ruby on Rails.

Installation :

  • Run git clone
  • cd to the app folder
  • Run bundle install
  • Run rails g rails_admin:install
  • Run rake db:migrate or bundle exec rake db:migrate
  • Run rails server
  • Open localhost:3000 on your browser

Feature:

  • Integrated with devise authentication, so visitor can create account for them.
  • Bootstrap-sass for styling
  • paperclip gem for uploading image
  • Using rails_admin gem for site admin ( New )
